    Concise, practical course outlines and recommended tools to build skills across web, backend, data, and DevOps.

    Beginner Web Development Backend & APIs Data & ML DevOps & Tools

    Beginner Path

    Intro to Programming (Python)
    Duration: 6 weeks • Prereq: none
    • Syntax, variables, control flow
    • Functions, modules, file I/O
    • Small projects: calculator, text parser
    Web Fundamentals (HTML & CSS)
    Duration: 4 weeks • Prereq: none
    • Semantic HTML, responsive layout, accessibility
    • CSS basics, Flexbox, Grid
    • Project: personal portfolio site
    Version Control (Git)
    Duration: 1–2 weeks • Prereq: none
    • Commits, branches, merge, remote workflows
    • Using GitHub/GitLab

    Web Development

    Frontend: JavaScript & Frameworks
    Duration: 8–10 weeks • Prereq: HTML/CSS
    • Vanilla JS: DOM, async, fetch
    • Framework focus: React (or Vue/Svelte)
    • State management, routing, testing
    Full-Stack Project
    Duration: 6–8 weeks • Prereq: frontend + backend basics
    • Build CRUD app with frontend + REST/GraphQL API
    • Deploy to a cloud provider (Netlify, Vercel, or similar)
    • CI, environment variables, basic security

    Backend & APIs

    Node.js / Express
    Duration: 4–6 weeks • Prereq: JS basics
    • Routing, middleware, authentication
    • Database integration (MongoDB/Postgres)
    • Testing and error handling
    Python (Flask / FastAPI)
    Duration: 4–6 weeks • Prereq: Python basics
    • REST APIs, async (FastAPI), validation
    • ORM usage, migrations, role-based auth

    Data & Machine Learning

    Data Engineering Essentials
    Duration: 6 weeks
    • ETL basics, SQL, data warehousing concepts
    • Streaming vs batch, basic pipelines
    Intro to ML with Python
    Duration: 8 weeks • Prereq: Python, math basics
    • NumPy, pandas, scikit-learn
    • Model evaluation, simple pipelines

    DevOps & Tools

    Essential Tools
    • Editor/IDE: VS Code, JetBrains
    • Version Control: Git + GitHub/GitLab
    • Containerization: Docker
    CI/CD & Deployment
    • CI: GitHub Actions, GitLab CI
    • Cloud: Vercel, Netlify, AWS/GCP/Azure basics
    • Infrastructure as Code: Terraform (intro)
    Observability & Testing
    • Testing: unit, integration, end-to-end
    • Monitoring: basic logging and metrics (Prometheus/Cloud)

    Learning Tips & Suggested Plan

    • Start small: build projects to solidify each concept.
    • Practice consistently and use version control for every project.
    • Combine theory with hands-on (tests, deployments, debugging).
